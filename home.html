Elsa's skills

初学前端工程师时,最先接触的是HTML、CSS随后就是JavaScript.

  • 

    HTML

    HTML的全称为hypertext markup language,即超文本标记语言。HTML是一种规范，一种标准，它通过标记符号来标记要显示的网页中的各个部分，可以告诉浏览器如何显示其中的内容，它是一种由浏览器解释执行的语言（如文字如何处理，画面如何安排，图片如何显示等）。其中的超文本就是指页面内可以包含图片、链接、甚至音乐等非元素文字元素.

  • 

    CSS

    css,层叠样式表（Cascading style sheets） 作用：给网页中的元素添加样式.

  • 

    JavaScript

    JavaScript是运行在浏览器端的一个脚本语言;(脚本语言既能编程又能直接解析执行的语言）JS的特点:基于对象和事件驱动的松散型的解释型语言.
